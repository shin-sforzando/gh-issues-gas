# gh-issues-gas

GitHub Issues Importer from Google Spreadsheet

- [How to](#how-to)
  - [Develop](#develop)
    - [Clasp](#clasp)

## How to
### Develop

```shell
$ npm install
```

And you have to enable "Google Apps Script API" at [G Suite Developer Hub](https://script.google.com/home/usersettings).

#### Clasp

```shell
$ npx run clasp login
```
